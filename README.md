# 📚 学科助教系统 Edu-TA

> **面向一流学科建设的学科垂类大模型与创新应用开发 — 助教端**

基于 LLM + RAG 技术的智能教学辅助系统，覆盖**智能备课、作业批改与辅导、学情精准洞察、学科知识库**四大核心模块，以 AI 赋能教学全流程，助力高校教师减负增效。

## ✨ 核心功能

### 📝 智能备课
- **AI 教案生成**：输入课程名称和章节，自动生成完整教案
- **RAG 增强**：从学科知识库检索教材内容，提升备课质量
- **结构化输出**：教学目标（知识/能力/素质）、教学活动设计、课后作业
- **灵活定制**：支持课时数、教学要求等自定义参数

### ✅ 作业批改与辅导
- **智能批改**：支持选择题、填空题、主观题、计算题、论述题
- **详细反馈**：得分、评语、优点/不足分析、改进建议、逐题分析
- **批量批改**：支持全班作业批量处理与成绩分布统计
- **智能出题**：针对薄弱知识点自动生成练习题

### 📊 学情精准洞察
- **知识掌握度**：雷达图展示各知识点掌握水平
- **趋势分析**：成绩变化趋势追踪
- **薄弱识别**：自动识别薄弱知识点
- **预警系统**：学习预警与重点关注名单
- **个性化建议**：AI 生成针对性学习建议
- **班级总览**：班级整体学情分析

### 📚 学科知识库
- **文档导入**：支持 PDF/TXT 教材导入
- **向量化存储**：自动分块与向量化（ChromaDB）
- **语义检索**：基于语义相似度的精准检索
- **多课程管理**：各课程知识库独立管理

## 🐳 Docker 一键启动（唯一方式）

> 本项目完全基于 Docker 运行，无需安装 Python、Node.js 等本地环境。

### 前置条件
- Docker 和 Docker Compose
- 一个 OpenAI 兼容的 API Key（DeepSeek / 讯飞星火 / OpenAI 等）

### 启动

```bash
# 构建并启动
docker compose up -d

# 访问 http://localhost:8080
# API 文档 http://localhost:8080/docs
```

### 常用命令

```bash
# 启动
docker compose up -d

# 查看日志
docker compose logs -f

# 停止
docker compose down

# 重启
docker compose restart

# 重新构建（更新代码后）
docker compose up -d --build
```

> **注意**：首次启动需要下载 Embedding 模型（~100MB），请确保网络通畅。

## 🔒 API 密钥安全说明

> 本项目所有 LLM API 密钥均采用**浏览器端本地存储（localStorage）**，仅保存在您当前使用的浏览器中。
> 
> - ✅ **密钥不会提交到代码仓库** — .env 文件已被删除并加入 .gitignore
> - ✅ **密钥不会上传到后端服务器** — 每次请求由前端读取 localStorage 中的密钥，以 HTTP 头发送至后端实时调用，后端不持久化存储
> - ✅ **密钥不会随项目分发** — 其他电脑或用户克隆本项目后无法获取您的密钥，需各自在浏览器中独立配置
> - ✅ **多设备独立** — 每台电脑、每个浏览器、每个账户都需要单独配置自己的 API Key
> 
> 您可以在「LLM API 配置」页面底部看到 🔒 安全提示确认密钥存储方式。
